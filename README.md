## Overview

This application is a mobile-first e-commerce platform built with Flutter, featuring product browsing, shopping cart functionality, and user authentication. The project was developed as coursework for the Programming Applications and Programming Languages (M30235) module.

## Features

- Product catalog with categories and collections
- Product detail pages with images and descriptions
- Shopping cart functionality
- User authentication
- Responsive mobile-first design
- Navigation between pages

## Getting Started

### Prerequisites

- Flutter SDK
- Dart
- Visual Studio Code or Android Studio
- Chrome browser (for web testing)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR-USERNAME/union_shop.git
cd union_shop
```

2. Install dependencies:
```bash
flutter pub get
```

3. Run the application:
```bash
flutter run -d chrome
```

Use Chrome DevTools (`F12`) and toggle device toolbar to view in mobile format.

## Running Tests

```bash
flutter test
```

## Project Structure

- `lib/` - Main application source code
- `test/` - Unit and widget tests
- `pubspec.yaml` - Project dependencies and configuration
